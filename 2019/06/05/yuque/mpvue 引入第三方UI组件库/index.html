mpvue 引入第三方UI组件库

Author: httishere Date: June 5, 2019  13:52:32

1、下载第三方UI组件包，将其dist文件夹复制至自己项目下

2、在需使用组件的页面json内引入需要的组件

3、直接在内面内使用组件

注意：
①、需要将微信开发工具的ES6转ES5选项选上，否则报错。

②、我项目出现的问题，原先我把第三方的dist放在项目文件的dist文件下，但是发现还是会报not found错误，然后转移到项目的static文件夹下就好了。
