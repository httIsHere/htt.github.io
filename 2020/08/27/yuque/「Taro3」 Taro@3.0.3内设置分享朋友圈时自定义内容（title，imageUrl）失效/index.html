「Taro3」 Taro@3.0.3内设置分享朋友圈时自定义内容（title，imageUrl）失效

Author: httishere Date: August 27, 2020  14:02:46

taro@3.0.3 内缺失相应的类型定义，在设置分享朋友圈自定义内容时导致自定义内容失效。

解决：
• 升级至 3.0.3 以上；
• 在global.d.ts内添加缺失的类型定义，并在需要分享朋友圈的页面配置文件内添加enableShareTimeline: true。

参考#7055: https://github.com/NervJS/taro/pull/7055

注：
设置自定义分享时 query 需要采用 object 的形式。
