「Less」使用总结

Author: httishere Date: 十二月 15, 2020  11:59:22 Category: css/Less

less 相关知识点

  • Less 是一门 CSS 预处理语言，它扩展了 CSS 语言，增加了变量、Mixin、函数等特性，使 CSS 更易维护和扩展。
  • Less 可以运行在 Node 或浏览器端。

Less 和 Sass 在语法上有些共性，比如下面这些：

  • 混入(Mixins)： class 中的 class；
  • 参数混入：可以传递参数的 class，就像函数一样；
  • 嵌套规则：Class 中嵌套 class，从而减少重复的代码；
  • 运算—：CSS中用上数学；
  • 颜色功能：可以编辑颜色；
  • 名字空间(namespace)：分组样式，从而可以被调用；
  • 作用域：局部修改样式；
  • JavaScript 赋值：在 CSS 中使用 JavaScript 表达式赋值。

安装使用，#using-less

一般项目内使用webpack作为打包工具，可以在webpack打包时进行 less 编译解析，需要安装less和less-loader。
